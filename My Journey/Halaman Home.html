Roller Coaster Of Life

Hai teman-teman perkenalkan nama saya Margaretha Ekaristi Yobella. Teman-teman saya biasa memanggil saya dengan panggilan Margaretha atau Retha. Saya adalah mahasiswa UNESA, Fakultas Teknik,Jurusan Teknik Informatika, Prodi S1 Teknik Informatika. Dengan NIM 18051204009.Sekarang saya sedang menempuh pendidikan di semester 4. Kali ini saya membuat website ini untuk memenuhi tugas pemograman web dari Bapak I Made Suartana, S.Kom., M.Kom. dan juga tentu untuk belajar serta mengembangkan kemampuan saya dalam bidang pemrogaman web. Tugas dari mata kuliah ini adalah membuat web bertemakan "My Journey" jadi disini saya akan menceritakan beberapa pengalaman saya selama hidup di dunia ini. Jika teman-teman tertarik untuk membaca,teman-teman bisa meng-klik beberapa opsi yang tersedia di web ini.

Hidup itu seperti roller coaster, terkadang kita ada dibawah dan bisa diatas. Maka dari itu, saya menggunakan judul Roller Coaster Of Life untuk memenuhi tugas pemograman web ini. Saya harap dengan saya membuat Roller Coaster Of Life tentang saya bisa menjadi bermanfaat bagi saya maupun para pembaca untuk menjadi pribadi yang semakin baik.

Selamat membaca. Salam hangat dari saya, Margaretha.
